# smart-chatbot-game

「心流對話局」是一款可直接部署到 GitHub Pages 的單頁式智慧聊天網頁遊戲。

## 遊戲特色

- 玩家在遊戲開始前選擇聊天方向與主題
- 可設定聊天機器人的個性
- 以對答方式進行六回合遊戲
- 根據玩家回應累積分數並解鎖不同結局
- 支援手機、平板與電腦瀏覽器
- 不需要安裝套件或後端伺服器

## GitHub Pages 發布方式

1. 將 `index.html`、`README.md` 與 `.nojekyll` 上傳到 Repository 的 `main` 分支根目錄。
2. 開啟 Repository 的 **Settings**。
3. 選擇左側 **Pages**。
4. 在 **Build and deployment** 中將 Source 設為 **Deploy from a branch**。
5. Branch 選擇 **main**，資料夾選擇 **/(root)**，再按 **Save**。
6. 等候 GitHub 完成部署。
